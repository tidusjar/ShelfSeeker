# Suggested Commands for shelfseeker

## Development
```bash
npm run dev        # Run in development mode with tsx
```

## Production
```bash
npm run build      # Compile TypeScript to dist/
npm start          # Run compiled version
```

## Project Info
- **Purpose**: IRC ebook search and download CLI tool for IRCHighway #ebooks channel
- **Tech Stack**: TypeScript, Node.js, irc-framework, prompts, adm-zip
- **Main Entry**: src/index.ts
- **IRC Server**: irc.irchighway.net:6667
- **Channel**: #ebooks
- **Search Command**: @search <query>
